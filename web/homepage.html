Pill Identifier

By: Konstantinos Moisakos

Project Description

The purpose of this project is to utilize machine learning and computer vision to identify pills. The target audience are healthcare workers or people that take many medications that need to quickly identify an ulabeled pill they have. Although a small-scale project, this provides proof of concept for a larger scale project in the medical field. This project also proves of the power of machine learning and how it can be used to improve vision processes that need to identify key objects. Machine learning and AI is the future of technology and this project is a small step in that direction. In order to use this application, navigate to the video page to upload a video or the webcam page to use live webcam.

Features

This project is a web application that uses YOLOV8 and OpenCV for the machine learning and computer visions aspects respectively. The model is trained on a custom dataset of 49 images using Roboflow. The user can upload a video of a pill or use live webcam to view pills. The model will return the name of the pill and provide a brief description of the pill. This webpage is built using flask.
